## <picture><img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="28"></picture> About me

Sou engenheiro **full-stack** brasileiro, focado em **construir software de aviação** de ponta a ponta — do pipeline de dados aeronáuticos até a tela no cockpit. Hoje toco sozinho a **Alfaero**, uma plataforma completa para pilotos: app **Flutter** (Android, iOS, Windows e Web) com cartas offline, METAR/TAF/NOTAM, navegação, *synthetic vision* e meteorologia animada — tudo sustentado por um **backend 100% serverless na AWS** (Lambda + API Gateway + Step Functions + DynamoDB).

No dia a dia, trabalho na **Superlógica** evoluindo um ERP imobiliário multi-tenant de larga escala (PHP/Zend legado) — cobranças, recebimento e regras de negócio críticas que rodam para milhares de imobiliárias.

Minha praia é **serverless-first**: prefiro pipelines pequenos, baratos e resilientes a monolitos. Opero **toda** a infra da Alfaero (AWS + Hostinger + Cloudflare) por ~US$49/mês, com dúzias de Lambdas, Step Functions e BFFs que ingerem cartas (AIRAC), METAR/TAF/NOTAM e procedimentos SID/STAR/IAP do DECEA e os entregam estruturados pro app.

Quando sobra tempo, brinco com **hardware** (ESP32) — de um tamagochi pra minha filha a um AMS caseiro pra impressora 3D.

> *"Do raster da carta ao cockpit — eu construo a stack inteira."*
